Weather App

A web-based application that provides current weather information and a five-day forecast. It utilizes HTML, CSS, JavaScript, and the ReactJS framework, along with an API to retrieve weather data.The app displays the current day's temperature, weather conditions, humidity, wind speed, and the high and low temperatures. Additionally, it presents a forecast for the next five days, including the high and low temperatures for each day. The app defaults to Houston, TX, but allows users to search for other locations. The background dynamically changes based on the time of day in the searched city. It shows a sunny sky during daylight hours and a night sky during nighttime.

Memory Game

This is a web-based game where users have to match pairs of symbols until all pairs are successfully matched. It is built using HTML, CSS, JavaScript, and the ReactJS framework. The objective of the game is to test and improve the player's memory skills. Players need to flip over two cards at a time, trying to find matching pairs by remembering the position of each symbol. Once a pair is matched, those cards remain face-up. The game ends when all pairs have been successfully matched.
